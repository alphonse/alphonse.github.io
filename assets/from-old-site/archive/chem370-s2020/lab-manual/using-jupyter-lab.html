CHEM 370 Lab Manual

Using Jupyter Lab

Option 1: Windows

  1. Click the magnifying glass in the bottom left corner of the screen.
  2. Search for anaconda prompt.
  3. Open the Anaconda Powershell Prompt that appears in the search results.
  4. Type jupyter-lab into the prompt and press enter. A web browser should open with a Jupyter window.

Option 2: Mac OSX / Linux

  1. Open a terminal application (Mac: CMD + Space then search Terminal and open Terminal.app).
  2. Type jupyter lab into the terminal prompt and press enter. A web browser should open with a Jupyter window.

Option 2: Using VCAT

Jupyter Lab and Octave are installed on WCUâs STEM virtual machine (STEM VCAT). You may use this as an option if you have trouble installing Octave and/or Jupyter on your own computer. Note, however, that this option will only work when you have internet access and can slower and more difficult than running the software on your own computer.

Logon to VCAT

  1. Logon to my.wcu.edu.

  2. Click Virtual Apps (VCAT) under the Quick Links section of my.wcu.edu.

  3. Hover over the STEM Specialized option and click the three dots that appear in the upper right corner. Choose Open in Client.

    You will need to install the VM Ware Horizon Client if you have not already done so (you must also keep the client up to date for it to work properly). Please visit Technology Commons if you have trouble installing or using the client.

Start Jupyter
  1. Click the magnifying glass in the bottom left corner of the screen.
  2. Search for anaconda prompt.
  3. Open the Anaconda Powershell Prompt that appears in the search results.
  4. Type jupyter-lab into the prompt and press enter. A web browser should open with a Jupyter window.
